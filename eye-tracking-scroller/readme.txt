INTRODUCTION
------------

An open source project that uses a webcam to scroll through a PDF.
    
    * To submit bug reports and feature suggestions, or to track changes:
        -https://github.com/shobhitvoleti/eye-tracking-scroller
        
REQUIREMENTS
------------

This module requires the following modules:

 * Tkinter (https://docs.python.org/2/library/tkinter.html)
 * OpenCV (https://pypi.python.org/pypi/opencv-python)
 * PyPDFOCR (https://pypi.python.org/pypi/pypdfocr)

 
 How To Install
 --------------

* Run command python setup.py build

* Wait for command to finish the build and run
  - python setup.py install 

* Execute application
